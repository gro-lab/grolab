Installation Guide

1

Download

Get the latest extension package from the button below or from GitHub Releases.

2

Extract

Unzip ai-browser-extension.zip to a permanent folder on your computer. Don't delete this folder.

3

Load in Chrome

Open chrome://extensions/ in Chrome. Enable Developer Mode (top right). Click Load Unpacked and select the extracted folder.

4

Configure API Key

Getting an API Key

Troubleshooting

Extension not loading? Check chrome://extensions/ for error messages.

AI not responding? Verify your API key is correct in settings. Check the background console for errors.

Actions not working? Some sites block content scripts. Try refreshing the page.

Also works on Microsoft Edge and other Chromium-based browsers.
